Tag List

  • main: wanneer je een main-tag plaatst, wil dat zeggen dat jouw hoofd informatie daar moet staan.
  • section: sections kan je zien als hoofdstukken van de main-tag.
  • ul: gebruik je wanneer de volgorde niet uit maakt
  • ol: gebruik je wanneer de volgorde wel uit maakt bijvoorbeeld bij een recept.
  • li: li-tag kan je zien als bulletpoints.
  • em: om je tekst italic te maken.
  • strong: om j tekst bold te maken.
  • a href: gebruik je om hyperlinks te plaatsen.
  • form: data dat je wil sturen. Je kan het zien als een enquete of mail met data.
  • button type: een knop die de user kan drukken bijvoorbeeld button type="submit". Als de user dit drukt wordt het ingevulde data verstuurd(zie form-tag).
  • fieldset: maakt een box waar er data is. kan gebruiken wanneer je je naam, nummer, e-mails enz. kan invullen.
  • legend: titel voor je fieldset.
  • figure: een box waar je foto's, illustraties kan plaatsen.
  • img src: gebruik je wanneer je de link van een foto wil plaatsen zodat e die op je website kan zien.
  • blockquote: gebruik je wanneer je een citaat wil toevoegen.
  • blockquote cite: hier zet je de link van de website waar je je citaat heb genomen.
  • input type: het maak een box waar de user iets kan typen.
  • label: de naam die je geef voor je input bijvoorbeeld naam(naam is je label):_____(underscores is waar je kan typen).
